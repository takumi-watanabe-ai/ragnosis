# Font Files

Place your font files here:

1. **Unica77-Regular.woff2** - For headers/titles
2. **CatalogueLL-Regular.woff2** - For hero subtext

Then uncomment the @font-face declarations in `app/globals.css`.

You can download these fonts from:
- The Index website (inspect and find font files)
- Font foundries that distribute these fonts
- Or use alternatives if you don't have licenses
